安装软件

  • manim本体：https://github.com/3b1b/manim
  • miktex: https://miktex.org/download
  • ffmpeg：http://ffmpeg.org bin目录添加环境变量
  • dvisvgm：https://sourceforge.net/projects/dvisvgm/ 添加环境变量

python依赖包

使用pip安装依赖，建议添加镜像源

  • pycairo https://www.lfd.uci.edu/~gohlke/pythonlibs/#pycairo

    ​ cp38对应的是python3.8版，cp对应3.7

    ​ pip install pycares‑3.1.1‑cp37‑cp37m‑win_amd64.whl

  • 其余依赖包，切换到manim安装目录，pip install -r requirements.txt

运行例子

python manim.py example_scenes.py SquareToCircle -pl
